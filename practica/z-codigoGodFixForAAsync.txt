codigo para evitar el problema con ciclo de vida en llamadas asincronas, cuando hacemos por ej una llamada a un endpoint, el componetne se destruye
y la llamada continua y se trata de resolver con un codigo de algo q ya no existe.

Este codigo lo soluciona, ejecuta la logica resultante solo si el componente sigue vivo:
import { AxiosResponse } from 'axios';
import { useEffect } from 'react';

export const useAsync = (
  asyncFn: () => Promise<AxiosResponse<any, any>>,
  successFunction: (data: any) => void,
  returnFunction?: () => void,
  dependencies: any[] = []
) => {
  useEffect(() => {
    let isActive = true;

    asyncFn().then((res) => {
      if (isActive) {
        successFunction(res.data);
      }
    });

    return () => {
      returnFunction?.();
      isActive = false;
    };
  }, dependencies);
};
